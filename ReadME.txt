Descrição Geral:
	Os arquivos de Testes devem estar na mesma pasta do programa que foi escrito em C utilizando de mascaras e shifts de bits.

Descrição das Funções:
	ethernetProtocol - Decodifica e exibe os campos do cabeçalho ETHERNET.
	ipProtocol - Decodifica e exibe os campos do cabeçalho IP.
	udpProtocol - Decodifica e exibe os campos do cabeçalho UDP.
	tcpProtocol - Decodifica e exibe os campos do cabeçalho TCP.

Execução:
	Para compilar o programa digite no terminal: gcc protoreader.c -o protoreader
	Para executar o mesmo digite: ./protoreader <nome do pacote.bin> 
			    Exemplo.: ./protoreader packet_udp.bin
